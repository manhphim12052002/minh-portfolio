# Writing UI Kit — Lumen

A long-form reading surface. Focused on one article at a time with a calm reader-mode feel. Includes an index and a single article view.

Components:
- `ReaderChrome.jsx` — minimal top bar with brand + TOC toggle
- `ArticleIndex.jsx` — chronological list
- `ArticleReader.jsx` — single-article layout with drop-cap, pulled quote, and end-of-article signoff
